# Concept
  
* 3-PartyPayment整合
  * 平台點數互相轉換
  * 點數可跨平台購物
* 平台會員
  * 建立消費者會員
    * 可結合 Amazon Member
    * 可結合 FB
    * 可結合 Line
    * 可結合 Google Member
    * 可結合 PChome
    * 可結合 Rakuten(樂天)
    * 可結合 Shopee(蝦皮)
    * 可結合 8591
  * 建立平台商家會員
    * 商家需具備合法營業許可
    * 提供規格化交易API/SDK給商家遵守
